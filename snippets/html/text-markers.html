abbr
In der Webprogrammierung werden viele Abkürzungen verwendet, wie HTTP oder UTF. Das abbr-Element (abbreviation) erlaubt uns, Abkürzungen in einem Text auszuzeichnen.
em
Die korrekte Verwendung semantischer HTML-Attribute ist sehr wichtig. Um einen Teil des Text zu betonen, gibt es das em-Element (emphasis).
i
Das Wort umfahren hat im Deutschen zwei gegenteilige Bedeutungen und kann wortwörtlich den Unterschied zwischen Leben und Tod bedeuten. Mit dem i-Element (idiomatic) zeichnen wie Text aus, der vom Rest abgehoben ist (etwa ein Wort, dass erwähnt, nicht benutzt wird).
dfn
Die Sprache JavaScript ist eine Multiparadigma-Sprache mit schwacher Typisierung. Das dfn-Element zeichnet eine Definition aus.
code
Um in JavaScript einen Text als Logausschrift auszugeben, benutzen wir die Anweisung console.log(text). Das code-Element verwenden wir für kleine Code-Schnipsel.
