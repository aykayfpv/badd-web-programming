Der Begriff Rostbratwurst ist komplex und stellt zunächst primär einen Oberbegriff über verschiedene Varianten von Würsten dar, die Sie grillen oder braten können. Die Bezeichnung hat keine einheitliche Gültigkeit, sondern definiert allgemein eine Brühwurst, die Sie sowohl mit mittelgrobem als auch grobem Brät gefüllt erhalten.
