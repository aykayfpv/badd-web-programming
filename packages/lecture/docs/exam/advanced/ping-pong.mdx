# Ping-Pong

Es soll ein Ping-Pong-Spiel erstellt werden, welches im Browser gegen einen
vom Computer gesteuerten Gegner spielbar ist.

## Mindestanforderungen

* Startbildschirm mit Option `Start`.
* Während des Spiels ein Button zum Beenden, woraufhin wieder der Startbildschirm erscheint.
* Zähler für die Punkte von Spieler und Gegner
* Ball muss nur in 45°-Winkeln abprallen.
* Gewinnt der Spieler oder Gegner, wird der Zähler entsprechend erhöht und
  es beginnt eine neue Runde.
