# Dynamische Anzeige

In Abhängigkeit einer Auswahl sollen verschiedene Bilder ein- oder ausgeblendet werden.

## Beschreibung

* 1 Auswahlfeld "Vorliebe" mit den Optionen `"Stillleben`, `Hörbuch`, `Kino`
* Je nach Auswahl soll ein Bild, Audio oder Video angezeigt werden.

## Mindestanforderungen

* Das Auswahlfeld soll auch die leere Option `Bitte auswählen` haben, hierbei
  wird nichts anderes angezeigt.
* Es kann immer nur 1 Option ausgewählt werden, keine Mehrfachauswahl. 
* Das Ein- und Ausblenden soll sofort beim Ändern der Auswahloption erfolgen.
* Wird das Audio oder Video gerade abgespielt, wenn auf eine andere Option
  gewechselt wird, soll das Audio oder Video pausiert werden.
