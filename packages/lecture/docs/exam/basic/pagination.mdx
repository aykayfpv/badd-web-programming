# Pagination

Ein Container mit vielen Elementen soll in Seiten unterteilt werden, wobei
auf jeder Seite nur eine Untermenge aller Elemente zu sehen sind.

## Beschreibung

* 1 Container mit 100 Elementen beliebigen Inhalts
* 1 Auswahlfeld für die Anzahl der Element pro Seite
* 1 Liste mit Links, jeweils ein Link pro Seite, um auf diese zu navigieren

## Mindestanforderungen

* Im Auswahlfeld für die Anzahl der Elemente pro Seite sollen folgende
  Optionen auswählbar sein: 1, 2, 5, 10, 20, __30__, 50, 100, 200.
* Initial sollen 20 Elemente pro Seite angezeigt werden.
* Änderungen im Auswahlfeld für die Anzahl der Elemente pro Seite sollen sofort
  angewandt werden. Bei jeder Änderung wird die aktuell angezeigte Seite auf
  `1` zurückgesetzt.
* Bei Klick auf einen Seitenlink soll auf die entsprechende Seite gesprungen werden,
  d.h. die Elemente dieser Seite angezeigt werden.
