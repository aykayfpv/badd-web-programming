# Pascalsches Dreieck

Es soll das [Pascalsche Dreieck](https://en.wikipedia.org/wiki/Pascal%27s_triangle)
für eine frei definierbare Anzahl an Zeilen angezeigt werden.

## Beschreibung

* Eingabefeld für die Anzahl der Zeilen
* Button zum Erzeugen des Dreiecks
* Anzeige des Pascalschen Dreiecks

## Mindestanforderungen

* Für jede erlaubte Eingabe müssen alle Zahlen des Pascalschen Dreieck den
  korrekte Wert haben.
* Die Dreiecksform muss erkennbar sein, jede Zahl der muss zwischen den
  entsprechenden 2 Zahlen der Zeile darüber stehen.

## Hinweise

Beschränken Sie die Eingabe in einer sinnvollen Weise.

Falls sehr große Zahlen unterstützt werden sollen, kann
[bigint](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/BigInt)
genutzt werden.
