# Kapitel 1 - HTML

HTML ist die Grundlage für Webseiten und beschreibt deren Struktur. Die Abkürzung
bedeutet HyperText Markup Language -- also eine Sprache zur Auszeichnung von
Textinhalten mit Strukur. Mittels HTML können wir Text etwa auszeichnen mit
Links, Listen oder Tabellen. 

HTML ist ein sogenannter Living Standard. Das bedeutet, dass zu keinem Zeitpunkt
eine Version veröffentlicht wird, sondern sich der Standard kontinuierlich
weiterentwickelt. Der Standard wird kann eingesehen werden unter

> https://html.spec.whatwg.org/multipage/

Die [MDN web doc](https://developer.mozilla.org/en-US/) von Mozilla sind ein
weitere nützliche Ressource, welche Webtechnologien dokumentiert und erklärt.
