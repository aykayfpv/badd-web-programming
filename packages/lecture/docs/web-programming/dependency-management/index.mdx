# Kapitel 6 - Dependency-Management

Bisher haben wir uns die nativen (eingebauten) Möglichkeiten von HTML, CSS und vor
allem von JavaScript und der DOM-API angeschaut. Manchmal sind diese ausreichend,
aber in der Praxis hat sich herausgestellt, dass der Einsatz von weiteren Tools
uns hilfreich bei der Entwicklung von Anwendungen und Webapps unterstützen kann.

Wir wollen uns daher in diesem Kapitel mit den Grundlagen des JavaScript-Ecosystem
vertraut machen. Vereinfacht gesagt bezieht sich der Begriff JavaScript-Ecosystem
auf verschiedene Programme, die wir zum Entwickeln verwenden und unseren Quellcode
in den finalen Stand überführen, der auf einem Server für den Produktivbetrieb
der Webseite hochgeladen wird.
