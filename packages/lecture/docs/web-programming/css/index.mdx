# Kapitel 2 - CSS

In diesem Kapitel schauen wir uns die Stylesheet-Sprache CSS an. CSS wird
verwendet, um Webseiten schön zu machen. Die Abkürzung CSS steht für
`Cascading Style Sheets`, eine Abkürzung, deren Bedeutung wir in Kürze verstehen
werden.

Der CSS-Standard wird von der W3C-Organisation verwaltet und besteht aus mehreren
Teilen für verschiedene CSS-Features. Er kann hier eingesehen werden:

> https://www.w3.org/Style/CSS/Overview.en.html

Neues Features für CSS werden in verschiedenen Stages eingeführt, wobei
Stage 0 nur eine Idee und in Stage 4 ein Feature Teil des CSS-Standards ist.
Eine Liste neuer Features und deren Status findet sich hier:

> https://cssdb.org/
