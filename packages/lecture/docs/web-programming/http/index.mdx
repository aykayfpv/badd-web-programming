# Kapitel 4 - HTTP

HTTP steht für `Hyper Text Transfer Protocol` und wird verwendet für die Übertragung
von HTML, der `Hyper Text Markup Language`.

Das HTTP-Protokoll wurde einfach gehalten und hat sich inzwischen als bewährtes
Mittel für die Kommunikation allgemein herausgestellt, nicht für den Austausch
von HTML-Dokumenten. Beispielsweise gibt es [HLS](https://en.wikipedia.org/wiki/HTTP_Live_Streaming),
was für `HTTP Live Streaming` steht und wobei Video-Streams per HTTP übertragen werden.

HTTP ist standardisiert per [RFC 9110](https://www.rfc-editor.org/rfc/rfc9110.html).
